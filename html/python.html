Цель реализации программы:
Познакомить слушателей с основами программирования на языке Python, сформировать соответствующие умения и навыки, необходимые для разработки программ на языке Python.
Слушатель, освоивший программу должен:
знать:

- основные понятия и термины в области программирования;
- базовые конструкции структурного программирования;
- возможности программирования на языке Python.

уметь:

- записывать алгоритмические структуры на языке программирования Python;
- решать прикладные задачи в Python;
- разрабатывать программы на языке Python.

владеть навыками:

- навыками разработки программ на языке Python.
