Цель реализации программы:
Развитие у слушателей навыков решения экспериментальных задач по физике, необходимых для участия в олимпиадах и конкурсах по физике различного уровня, а также развитие навыков использования IT-технологий при решении физических задач и в процессе проектно-исследовательской деятельности.
Слушатель, освоивший программу должен:
знать:

- виды экспериментальных задач по физике и методику их решения;
- методику решения экспериментальных олимпиадных задач по физике;
- методологию учебно-исследовательского проекта по физике;
- классы точности измерительных приборов;
- специфику обработки результатов эксперимента;

уметь:

- решать экспериментальные задачи по физике разного уровня сложности;
- рассчитывать погрешности физических измерений;
- прогнозировать, анализировать и обобщать результаты физических экспериментов;
- оформлять результаты физического эксперимента;
- использовать IT-технологии при решении физических задач и в процессе проектно-исследовательской деятельности;
- пользоваться различным физическим оборудованием с соблюдением правил техники безопасности;

владеть навыками:

- постановки физического эксперимента;
- решения олимпиадных и проектно-исследовательских задач по физике;
- применения IT-технологий при решении физических задач.
Материально-техническая база
Обучение проводится в аудиториях, оснащенных мультимедийным оборудованием.
При проведении занятий используется следующее физическое оборудование:
Комплект ГИА-лабораторий по физике-2023
ФГОС лаборатория для сдачи ГИА и ЕГЭ цифровая:
- комплект для демонстраций по механике на воздушной подушке КДМВ;
- пистолет баллистический;
- машина электрическая обратимая (двигатель-генератор);
- лабораторный набор «Электромагнит разборный с деталями»;
- модель электродвигателя (разборная) лабораторная
