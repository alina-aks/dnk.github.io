Цель реализации программы:
Познакомить слушателей с основными правилами создания веб-страниц, инструментами для создания веб-сайтов, подготовки различного вида контента для публикации, сформировать умения и навыки работы с языком разметки HTML, основами каскадных таблиц стилей (CSS), языка программирования (JS) и его библиотек, создания и размещения в сети Интернет web-страниц.
Слушатель, освоивший программу должен:
знать:

- основную терминологию в области web-технологий;
- базовую структуру web-страницы;
- основы проектирования дизайна web-страницы;
- способы создания веб-сайтов (конструкторы, CMS, кодинг);
- основы синтаксиса HTML, CSS, JS;
- основные принципы дизайна сайтов.

уметь:

- грамотно использовать язык разметки HTML;
- добавлять на web-страницу изображения, списки, favicon, таблицы;
- использовать каскадные таблицы стилей (CSS) для оформления страниц;
- создавать небольшие скрипты с использованием языка JS;
- добавлять элементы веб-интерфейса с использованием Jquery UI, Bootstrap
- создавать web-страницы различного вида, в том числе адаптивные;
- размещать web-страницы и контент в сети Интернет;
- пользоваться популярными редакторами web-дизайнера и разработчика.

владеть навыками:

- самостоятельного создания типового web-сайта.
Материально-техническая база
Для проведения занятий используется компьютерный класс, оборудованный компьютерами с выходом в сеть Интернет, с соответствующим программным обеспечением (текстовым редактором Notepad++, Sublime Text, Atom или VS Code, браузерами Mozilla Firefox или Google Chrome), оснащенный маркерной доской, медиапроектором, экраном.
