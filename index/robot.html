Робототехника на платформе Arduino (углубленный уровень)

Класс :

7 / 8 / 9 / 10

Количество учеников:

14

Трудоёмкость программы: 72 часа


День недели:

суббота 15:00 – 17:20

Аудитория:

3201
Цель реализации программы:
Совершенствование компетенций слушателей в области робототехники, через их вовлечение в проектную деятельность по конструированию, программированию и управлению различными видами роботов, созданных на базе платформы Arduino.
Слушатель, освоивший программу должен:
знать:

- основы создания робототехнических устройств;
- понятия, компоненты и принципы работы с электроникой и электротехникой;
- основные правила и приемы проектирования электронных систем разной сложности;
- правила техники безопасности при работе с инструментами, электрическими приборами и робототехническими наборами;
- основы программирования робототехнических устройств;
- основные правила при работе с 3D-принтерами различных моделей;
- основные методы разработки трёхмерных моделей.

уметь:

- создавать электрические системы, схемы, устройства;
- пользоваться различными видами датчиков при проектировании робототехнических устройств;
- создавать и программировать роботов на платформе Arduino;
- самостоятельно решать технические задачи в процессе конструирования роботов;
- демонстрировать технические возможности созданных проектов;
- решать технические задачи с помощью методов прототипирования и конструирования;
- создавать трехмерные компьютерные модели в программе «Компас-3D».

владеть навыками:

- сборки и программирования робототехнических устройств разной сложности с использованием платформы Arduino;
- программирования собранных конструкций под различные задачи;
- визуализации проектируемых объектов в программе «Компас-3D».

Материально-техническая база
Для реализации программы используется кабинет робототехники, оснащенный мультимедийным оборудованием, интерактивной доской, радиомикрофонами, звуковыми колонками. При проведении обучения используется следующее оборудование:

- интерактивная доска с выходом в интернет;
- ноутбуки ASUS TUF Gaming A15 с предустановленным пакетом САПР Компас 3D;
- наборы для конструирования «Эвольвектор» (базовый и ресурсный);
- наборы для конструирования «Амперка»;
- базовые наборы Tetrix Prime 44320 с дистанционным управлением;
- базовые наборы Tetrix Prime 44321 с автономным управлением;
- 3D-принтеры: Flashforge Dreamer; Wanhao Duplicator 6; 3D-принтер MZ3D-360; FlashForge Adventurer 3;
- пластик для 3D-печати, мастихин и пр.;
- паяльник импульсный «REXANT Профи»;
- паяльные станции: ELEMNT 879; AKTAKOM ASE4205;
- микроконтроллерные платформы (Arduino Uno R3, Arduino nano 3.0, Arduino Mega 2560 R3 и др.);
- платы для Arduiono (Enternet Shield Sensor Shield V5.0);
- вибромотор плоский для Arduino
- датчик-магнетометр LIS3MDL для Arduino
- сервопривод TowerPro для Arduino
- транзистор полевой AMP-X015
- биполярные транзисторы
- модуль Bluetooth HC-06
- модуль Wi-Fi esp32 dev kit с керамической антенной
- раздаточный материал.
